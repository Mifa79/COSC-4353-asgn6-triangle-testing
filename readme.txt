## How to run the program:
- In Command Line, cd to the target folder of the maven project (maven-demo folder)
- Use the command "mvn compile" to compile the program
- Use the command "mvn test" to run the tests
- Tests run: 33, Failures: 0, Errors: 0, Skipped: 0
- The test has 100% branch coverage
- To see the full JavaCoCo Report in details, go to "maven-demo/target/my-reports" and open the "index.html" file by a browser.
